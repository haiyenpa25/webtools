---
description: Chạy WebTools CMS server bằng file run.bat
---

Chạy file `run.bat` để khởi động WebTools CMS server tại http://localhost:3000.

// turbo

1. Chạy file run.bat để khởi động server:
```powershell
Start-Process -FilePath "c:\xampp\htdocs\Webtools\run.bat"
```

2. Xác nhận hoàn tất
Server đang khởi động trong cửa sổ mới. Truy cập http://localhost:3000 để vào dashboard.

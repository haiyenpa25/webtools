---
description: Tích hợp đồng thời Antigravity Awesome Skills, Superpowers framework, và TrustGraph Context Core
---

Quy trình này (workflow) đóng vai trò như một "câu lệnh thần thánh", thực hiện tự động cài đặt 3 công cụ mạnh mẽ: Antigravity Awesome Skills, Superpowers, và TrustGraph vào dự án của bạn để bắt đầu phát triển nhanh chóng.

// turbo-all

1. Cài đặt **Antigravity Awesome Skills** thông qua NPX (mang lại hơn 1,329 agentic skills).
```bash
npx -y antigravity-awesome-skills@latest
```

2. Cài đặt **Superpowers framework** qua tiện ích mở rộng của Gemini CLI.
```bash
gemini extensions install https://github.com/obra/superpowers
```

3. Ghi đè thiết lập context platform **TrustGraph** (bỏ qua đòi hỏi API keys, sử dụng mô hình nội bộ và storage cục bộ).
```bash
npx -y @trustgraph/config
```

4. Xác nhận hoàn tất
Tất cả các công cụ đã được chạy và tích hợp vào không gian làm việc của bạn! Bạn có thể bắt đầu sử dụng các skills mới ngay lập tức.
